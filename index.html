Introducing EIRI

The vision.

My art skills are dreadful.
But this was my vision.

The core idea was,

  • Multiple alarms able to be set
  • Next alarm displayed for comfort knowing my alarm is set
  • Anti blue light display to stop circadian rhythm disruption
  • Blackout modes no screen display or just next alarm displayed

What's included

The features.

Warm Night-Friendly Display

Soft amber LED display that's easy to read in the dark without harsh blue light that disrupts sleep.

Blackout Mode

Three display settings: display fully on, display off with next alarm visible, or completely off for total darkness.

Multi-Alarm System

Set up to 5 different alarms for people who prefer multiple wake-up times or reminders.

Central Control Button

Large recessed centre button to snooze, stop, or hold to set alarms. Designed to be easy to press when half asleep.

Simple Adjustment Buttons

Two arrow buttons for quick adjustments when setting alarms or changing the time.

Volume Dial

Side dial to quickly adjust alarm volume without navigating any menus.

Next Alarm Display

The screen shows your next scheduled alarm underneath the main clock for reassurance before you sleep.

Clear Alarm Indicator

A simple bell icon shows when an alarm is active so you know your wake-up is set.

USB-C Charging

Modern USB-C connection for simple charging and compatibility with common cables.

Thin Bed-Safe Design

Ultra-thin circular design so it can sit comfortably on a bed like a phone, if you don't use a bedside table.

Anti-Slip Base

Rubber base keeps the device stable on bedside tables or beds.

Distraction-Free Design

No apps, notifications, or smart features. Just time, alarms, and sleep-friendly functionality.
